Я выполнила лабораторную работу по созданию многостраничного приложения Student Portal на React + TypeScript с использованием React Router v6. В процессе работы я настраивала маршруты, сделала общий Layout с <Outlet />, навигацию через <Link /> и реализовала динамические страницы курсов с параметрами маршрута и query-параметрами. Также добавила страницу 404 для несуществующих маршрутов.

Во время работы я спрашивала про структуру папок, чтобы убедиться, что всё расположено правильно и не будет ошибок. Также возникали ошибки TypeScript, связанные с регистром файлов и импортами модулей — я их исправила, проверив имена файлов и обновив экспорты.

После всех исправлений приложение работает корректно: маршруты отображаются без перезагрузки страницы, динамические страницы курсов показывают данные по ID, а сортировка курсов через query-параметры работает.

Работа завершена, проект готов к запуску и отправке на GitHub.
